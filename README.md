# 🇲🇩 package-locale-mol (ISO 639-3)

Пакет цифровой локализации и транслитерации для молдавского языка на кириллической графике в соответствии с международным стандартом **ISO 639-3 (код: `mol`)**. Проект разработан под эгидой «Лумина Стандард» и предназначен для интеграции в современные веб-приложения, AI-оркестраторы и автономные системы автоматизации.

---

## ✨ Особенности / Спецификация

- **Полное соответствие ISO 639-3 (`mol`)**: Точная кодировка системных компонентов, дат, месяцев и интерфейсных экшенов.
- **Корректная кириллическая аффриката**: Полная поддержка специфичных фонем и знаков (например, букв **Ӂ / ӂ** в операциях типа `Штерӂе`, `Инреӂистраре`).
- **Интегрированный двунаправленный конвертер**: Встроенный модуль транслитерации `Latin ↔ Кириллица (Limba moldovenească)` для динамической обработки текстовых потоков.

---

## 📂 Структура локализации / Све деспре локализаре

Пакет включает в себя упорядоченные словари для системных настроек, авторизации, медиа-файлов и сетевых ошибок:

```json
{
  "locale": "mol",
  "meta": {
    "language_name": "Молдовеняскэ (Кириликэ)",
    "publisher": "Лумина Стандард"
  },
  "months": { "january": "Януарие", "february": "Февруарие", ... },
  "days": { "monday": { "full": "Лунь", "short": "Lu" }, ... },
  "actions": { "save": "Сальвязэ", "delete": "Штерӂе", "edit": "Модификэ" }
}
🚀 Установка и использование / Инсталаре ши утилизаре
​1. Подключение словаря локализации
​Импортируйте JSON-конфиг в ваш интернационализационный движок (i18next, Remix/Vite контекст или кастомный провайдер):
import molLocale from './locales/mol.json';

console.log(molLocale.notifications.welcome); // "Бине аць венит!"
2. Использование компонента транслитератора
​В состав пакета входит интерактивный веб-компонент конвертера (MoldovanConverter), разработанный на React и Tailwind CSS, поддерживающий автоматическую обработку диграфов (cea, cia, che, gi -> ӂи и др.).
​🛠 Разработка и сборка / Дезволтаре ши Сембларе
​Проект использует менеджер пакетов pnpm.
# Установка зависимостей
pnpm install --no-frozen-lockfile

# Запуск в режиме разработки
pnpm run dev

# Сборка продакшн-версии (Vite / Cloudflare Pages)
pnpm run build
